Первый запрос формирует отчет по обслуживанию клиентов. В отчете выводится информация о покупателе, количество его банковских счетов, число оформленных накладных, общее количество и стоимость всех товаров по накладным, число оплаченных накладных, фактически полученный доход по входящим платежным поручениям, а также количество и стоимость товаров в неоплаченных и аннулированных накладных. Все суммы должны быть приведены к рублям по курсу на дату продажи или оформления документов.
Второй запрос предназначен для выявления покупателей, превысивших среднее количество накладных. В результате должны выводиться данные покупателя, количество оформленных накладных, процент превышения среднего значения, количество оплаченных и количество аннулированных накладных.
Третий запрос формирует отчет по товарам максимального спроса. В нем указываются артикул, наименование товара, производитель, текущий остаток на складе, количество накладных, в которых товар присутствует, суммарное количество товара во всех накладных, суммарно проданное количество в оплаченных накладных, количество товара в неоплаченных и аннулированных накладных.
Четвёртый запрос предназначен для анализа товаров, цена которых не изменялась за последний месяц. В отчете выводятся артикул, название, производитель, остаток товара, валюта цены, текущая цена, текущая цена в рублях, количество накладных, суммарное количество товара во всех накладных, количество проданного товара, а также количество товара в неоплаченных и аннулированных накладных.
Пятый запрос формирует статистику продаж по месяцам выбранного года и должен содержать ровно 12 строк. Для каждого месяца выводятся его название и номер, количество проданных товаров, сумма продаж, самый популярный товар месяца с указанием количества и суммы продаж, самый доходный товар месяца, а также количество различных товарных позиций, проданных в этом месяце
